  This Jenkins plug-in collects the CSSLint (http://csslint.net/) analysis results of the project
  modules in Checkstyle format and visualizes the found warnings.
 
  This is a fork/specialization of the checkstyle plugin. See: http://wiki.jenkins-ci.org/x/GYCGAQ
